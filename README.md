# Number Guessing Game

Project URL: https://roadmap.sh/projects/number-guessing-game

A simple command line game where you try to guess a random number between 1 and 100. You can choose the difficulty level based on how many guesses you want. Try your luck and see if you can guess the number!

## How to Run

1. **Clone the repository:**

   ```
   git clone https://github.com/a-yasmin/number-guessing-game.git
   ```

2. **Navigate into the project directory:**

   ```
   cd number-guessing-game
   ```

3. **Run the game using Node.js:**

   ```
   node index.js
   ```

That's it! Follow the on-screen prompts to play the game.
